# Dashboard Statistics Service

## Overview

The dashboard displays various statistics about the e-commerce platform including order counts, user counts, and product inventory levels.

## Requirements

1. Display total number of orders
2. Show count of orders by status
3. Display total active users count
4. Show product count by category
5. Statistics must be up-to-date on dashboard refresh

## Constraints

1. Dashboard must load quickly (< 500ms)
2. Statistics are displayed as numeric counts only
3. No need to show individual record details on dashboard
