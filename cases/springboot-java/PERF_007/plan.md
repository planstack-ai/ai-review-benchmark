# Sales Analytics Service

## Overview

The analytics dashboard requires aggregated sales data including top products, revenue by category, and sales trends over time.

## Requirements

1. Calculate total revenue by product category
2. Identify top 10 best-selling products
3. Generate monthly sales trends
4. Support date range filtering
5. Handle millions of order records efficiently

## Constraints

1. Analytics queries must complete within 5 seconds
2. Must not cause memory issues on large datasets
3. Data should be calculated from order_items table
4. Must support real-time analytics without pre-aggregation
