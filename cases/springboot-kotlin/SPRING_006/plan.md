# SPRING_006: Order and Payment Service Integration

## Overview

Implement OrderService that handles order creation and interacts with PaymentService to process payments.

## Requirements

1. OrderService should create orders and process payments
2. PaymentService should handle payment processing and update order status
3. Services should collaborate to complete the order workflow
4. Proper transaction management across services

## Constraints

- Use Spring Boot 3.2+ dependency injection
- Follow Kotlin coding conventions
- Maintain service layer separation
- Ensure proper error handling
