# アクティブユーザー一覧取得

## 要件

1. アクティブなユーザーの一覧を取得する
2. 最終ログインが30日以内のユーザーのみを含める
3. 作成日の降順でソートする

## 使用すべき既存実装

- `User.active` - アクティブユーザーを取得するスコープ（deleted_at が nil かつ suspended が false）
- `User.recently_logged_in(days)` - 指定日数以内にログインしたユーザーを取得するスコープ

## 注意事項

- アクティブの定義は「削除されていない かつ 停止されていない」
- 既存スコープを組み合わせて使用すること
