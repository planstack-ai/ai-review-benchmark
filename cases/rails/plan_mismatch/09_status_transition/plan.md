# 注文キャンセル処理

## 要件

1. 注文をキャンセルする
2. **キャンセル可能なステータスは `pending` と `confirmed` のみ**
3. `shipped` 以降の注文はキャンセル不可
4. キャンセル時は在庫を戻す

## 使用すべき既存実装

- `Order#cancellable?` - キャンセル可能かを判定
- `Order#cancel!` - キャンセル処理（ステータス変更 + 在庫戻し）
- `Order.cancellable_statuses` - キャンセル可能なステータス一覧

## 注意事項

- ステータス遷移のルールは厳密に守ること
- `confirmed` からもキャンセル可能（発送前なら可）
