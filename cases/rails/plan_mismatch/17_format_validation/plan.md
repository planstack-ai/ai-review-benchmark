# 電話番号登録サービス

## 概要
ユーザーの電話番号を登録する。

## 要件

1. 電話番号を登録する
2. 電話番号の形式は **ハイフンあり**（例: 090-1234-5678）
3. 既存のバリデーションを使用する

## 使用すべき既存実装

- `User#phone_number=` - 電話番号設定
- `User::PHONE_FORMAT` - 電話番号の正規表現（ハイフンあり形式）
- `PhoneFormatter.normalize(number)` - 電話番号を正規化（ハイフンあり形式に変換）

## 注意事項

- 入力された電話番号は正規化してから保存すること
- ハイフンなしで入力されてもハイフンあり形式に変換する
