# ユーザープロフィール表示

## 要件

1. ユーザーのプロフィール情報を整形して返す
2. 表示名は `nickname` を優先、なければ `full_name` を使用
3. **自己紹介文（bio）は任意項目** - 未設定の場合は空文字を返す
4. 住所情報も任意 - 未設定の場合は「未設定」と表示

## 使用すべき既存実装

- `User#display_name` - 表示名を取得（nickname優先でfull_nameにフォールバック）
- `User#bio_or_default` - 自己紹介文を取得（未設定なら空文字）

## 注意事項

- nilの場合のハンドリングを適切に行う
- 任意項目でNoMethodErrorが発生しないよう注意
