# 商品一覧ページネーション

## 要件

1. 商品一覧をページネーションで取得する
2. **ページ番号は1始まり**（1ページ目、2ページ目...）
3. 1ページあたり20件
4. 指定ページの商品と総ページ数を返す

## 使用すべき既存実装

- `Product.page(page_number)` - 指定ページの商品を取得（1始まり）
- `Product.per_page` - 1ページあたりの件数（20）
- `Product.total_pages` - 総ページ数

## 注意事項

- ページ番号は **1始まり**（0始まりではない）
- 不正なページ番号（0以下、総ページ数超過）は1ページ目にフォールバック
