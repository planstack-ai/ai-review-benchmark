# 商品検索サービス

## 概要
複数条件で商品を検索する。

## 要件

1. 公開中の商品を検索する
2. 価格範囲で絞り込む（最小価格〜最大価格）
3. 在庫ありの商品のみ表示
4. 作成日時の新しい順にソート

## 使用すべき既存実装

- `Product.published` - 公開中の商品
- `Product.price_between(min, max)` - 価格範囲
- `Product.in_stock` - 在庫あり
- `Product.newest_first` - 新しい順

## 注意事項

- 全ての条件を正しく組み合わせること
- スコープを正しくチェインすること
