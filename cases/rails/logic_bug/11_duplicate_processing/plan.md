# ポイント付与サービス

## 概要
注文完了時にユーザーにポイントを付与する。

## 要件

1. 注文金額の1%をポイントとして付与する
2. ポイント付与は1回のみ行う（重複付与を防ぐ）
3. 付与したポイント数を返す

## 使用すべき既存実装

- `Order#total_amount` - 注文金額
- `Order#points_granted?` - ポイント付与済みフラグ
- `Order#mark_points_granted!` - ポイント付与済みにする
- `User#add_points(amount)` - ポイントを加算

## 注意事項

- 同じ注文に対して二重にポイントが付与されないようにすること
