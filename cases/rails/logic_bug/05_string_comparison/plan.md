# ステータスチェックサービス

## 概要
注文のステータスに基づいてキャンセル可否を判定する。

## 要件

1. 注文のステータスが "pending" または "confirmed" の場合のみキャンセル可能
2. "shipped" または "delivered" の場合はキャンセル不可
3. 大文字小文字を区別せずに判定する（DBの値が大文字で保存されている可能性があるため）

## 使用すべき既存実装

- `Order#status` - 注文ステータス（文字列）

## 注意事項

- ステータス値の大文字小文字を考慮すること
- 想定外のステータス値の場合はキャンセル不可とすること
