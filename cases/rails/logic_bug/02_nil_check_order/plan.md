# ユーザープロフィール表示サービス

## 概要
ユーザーのプロフィール情報を取得し、表示用データを構築する。

## 要件

1. ユーザーIDからユーザー情報を取得する
2. ユーザーが存在しない場合は nil を返す
3. ユーザーの所属会社情報も含めて返す（会社に所属していない場合は会社名は "個人" とする）

## 使用すべき既存実装

- `User.find_by(id:)` - ユーザーを取得（見つからない場合は nil）
- `User#company` - 所属会社（belongs_to, optional）
- `Company#name` - 会社名

## 注意事項

- NullPointerException（NoMethodError）が発生しないよう適切にnilチェックすること
